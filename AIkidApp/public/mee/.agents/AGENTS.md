# Project Rules

- **Development Port & URL**: Always serve and open the web application at `http://localhost:8000`.
- **Lobby Server**: The main server is located at `d:\StoryMee\new-lobby-page\server.js` and runs on port `8000`. It routes `/mee` requests to the SVG customizer page.
- **Do Not Use Port 3009**: Never run the local server on port `3009` or navigate to `http://localhost:3009`.
- **Open Web ('mở web')**: When the user requests to "mở web" (open web), always launch and open the web application starting from the lobby homepage at `http://localhost:8000`, and never use port `3009`.
